# lib-navbar

页面导航模块

客户端页面全屏展示文档：[APP协议](https://ones.ai/wiki/#/team/Tnb2S5Qj/space/YVQyvvRX/page/JcrWVQMk) => 打开H5页面跳转

## 安装

```shell
npm install @jyb/lib-navbar --save
```

## 开发

```shell
npm run dev
```

## 构建

```shell
npm run build
```

## 测试

```shell
npm test
```

## 覆盖率

```shell
npm run test:cover
```
