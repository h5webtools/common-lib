# BI,MTA 业务数据上报

提供`添加自定义数据属性上报`和`调用API上报`两种方式上报，可以自由选择

[更多文档](http://doc.fe.jyb.com/book/bimta/index.html)

## 安装

```shell
npm install @jyb/lib-bimta --save
```

## 开发

### 调试

```shell
npm run dev
```

### 构建

```shell
npm run build
```

### 测试

```shell
npm test
```

### 覆盖率

```shell
npm run test:cover
```
