# lib-tracker

前端数据采集，目前包括错误信息

## 文档

[数据平台:事件上报规范](http://wiki.jtjr.com/doku.php?id=%E6%95%B0%E6%8D%AE%E5%B9%B3%E5%8F%B0:%E4%BA%8B%E4%BB%B6%E4%B8%8A%E6%8A%A5%E8%A7%84%E8%8C%83)

## 安装

```shell
npm install @jyb/lib-tracker --save
```

## 开发

```shell
npm run dev
```

## 构建

```shell
npm run build
```

## 测试

```shell
npm test
```

## 覆盖率

```shell
npm run test:cover
```
