# 九宫格抽奖
- [详细使用方法见doc目录下的`QuickStart.md`](http://git.jtjr.com/h5_web_grp/common-lib/blob/master/packages/lib-lottery/doc/QuickStart.md)

## 安装

```shell
npm install @jyb/lib-lottery --save
```

## 开发

```shell
npm run dev
```

## 构建

```shell
npm run build
```
