# lib-qrcode

二维码生成模块，兼容IE6~10, Chrome, Firefox, Safari, Opera, Mobile Safari, Android, Windows Mobile, ETC.

## 安装

```shell
npm install @jyb/lib-qrcode --save
```

## 开发

```shell
npm run dev
```

## 构建

```shell
npm run build
```

## 测试

```shell
npm test
```

## 覆盖率

```shell
npm run test:cover
```
